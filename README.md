# SprayItForward
A shell script to generate a list of passwords to use in password sprays based on common season and year combinations.

## Setup
```shell
git clone
```
Give it execute permissions:
```shell
$ chmod +x sprayitfwd.sh
```

## Usage
```shell
$ ./sprayitfwd.sh
Autumn24
Autumn2024
Autumn24!
Autumn2024!
Fall24
Fall2024
Fall24!
Fall2024!
Summer24
Summer2024
Summer24!
Summer2024!
```
